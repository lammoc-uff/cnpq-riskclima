# filtered_catalog

This directory stores filtered catalog tables and catalog comparison outputs generated from the original CMIP6 provider catalogs.

## Main outputs

- ``catalog_cnpq_aws.csv`` — filtered AWS catalog
- ``catalog_cnpq_google.csv`` — filtered Google catalog
- ``catalog_diff_google_vs_aws.csv`` — entries available in Google but not in AWS after filtering
- ``catalog_diff_aws_vs_google.csv`` — entries available in AWS but not in Google after filtering

## Notes

- These files are generated by the catalog comparison workflow.
- They are useful for checking provider differences before starting downloads.
- Output names should remain consistent with the comparison script and project configuration.
