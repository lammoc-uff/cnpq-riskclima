# downloads

This directory stores the downloaded and processed CMIP6 outputs generated by the workflow.

## Main outputs

- per-group folders organized by ``source_id/experiment_id/table_id/variable_id/grid_label``
- ``catalog_group.csv`` — filtered catalog for a specific group
- ``group_log.csv`` — processing log for a specific group
- ``member-{member_id}.zarr`` — local Zarr store for one ensemble member
- ``ensemble_all.zarr`` — stacked ensemble output
- ``ensemble_mean.zarr`` — mean ensemble output
- ``log_download_results.csv`` — global processing log

## Notes

- These files are created automatically by the download pipeline.
- Output names and paths should remain consistent with the configuration defined in ``scripts/src/config.py``.
- Depending on project policy, large outputs may be better stored outside version control.
