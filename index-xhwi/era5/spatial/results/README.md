# results

This directory stores the output datasets generated by the XHWI workflow.

## Main outputs

- ``xhwi_era5_1950-2024_br_month_{m}_zarr_store.zarr`` — hourly XHWI results for each processed month
- ``xhwi_era5_1950-2024_br_diary_ind_prod_{m}_zarr_store.zarr`` — daily aggregated indicators for each processed month
- ``xhwi_era5_1950-2024_br_month_ind_prod_{m}_zarr_store.zarr`` — monthly aggregated indicators for each processed month
- combined daily NetCDF files — merged daily outputs generated during postprocessing
- combined monthly NetCDF files — merged monthly outputs generated during postprocessing

## Notes

- These files are created automatically by the processing scripts.
- Output names and paths should remain consistent with the configuration defined in ``src/config/settings.py``.
- Large output files may be better stored outside version control, depending on repository policy.
