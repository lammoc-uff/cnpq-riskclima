# index-xhwi

This directory references the main repository ``index-xhwi`` of the Extreme Heatwave Index ``XHWI`` used within the ``cnpq-riskclima`` project.

## What the project does

The XHWI repository provides the workflow and documentation required to compute the Extreme Heatwave Index from climate datasets.

## Why the project is useful

This index supports the analysis of heatwave-related climate extremes and contributes to the RiskClima objective of identifying vulnerable areas and climate-related hazards across Brazil.

## How users can get started with the project

The main development, codebase, and documentation are maintained in the dedicated XHWI repository:

- ``index-xhwi``
- ``https://github.com/lammoc-uff/index-xhwi``

## Where users can get help with your project

For usage details, installation instructions, and technical documentation, please refer to the main XHWI repository.
